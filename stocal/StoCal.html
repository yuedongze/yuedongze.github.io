Sony Network Camera Storage Calculator

Storage Calculator User Instruction

This storage calculator is dedicated for calculating storage requirement and bitrate consumption of Sony's camera products.

To use the calculator, first please choose the camera model that you want to use in the calculation on the bottom. Once you click the camera profile, the page will automatically load the corresponding data and enable mode selection.
Also, we have provided a customized camera profile feature so that you can load any camera database written in the corresponding JSON format into the storage calculator and perform calculation. In order to load the database, simply click the Choose File button below "Manually Load Camera Database", select the corresponding database file and click "Read".

After choosing the camera model, the page should automatically load the camera data and enables the user to add camera options.

When you choose the camera data, the page automatically added one input line for input for you. You can find different camera modes, images, bitrate options and more control parameters in the input fields.
Also, you can choose your video settings (NTSC or PAL) below the camera model name. All the input fields will automatically update its value based on your video mode.
The page will automatically update the bitrate and storage information for every input line and for total usage. Please add the lines as much as you need, and the page will automatically take account into all available lines.

An input line can also be removed by simply clicking the remove button on top of the corresponding line. Also there is a Clear All option besides the "Add New Option" button, which simply allows you to remove all lines.

On the bottom of the page there will be a simple calculation field. You can enter your current storage information or estimated time of calculation to obtain how much storage and how long time your situation requires.

There is also a printer friendly page option on the bottom of the page. Simply clicking the button will open up a new page with no redundant formatting and can be easily printed.
